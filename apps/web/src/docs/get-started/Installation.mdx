# Installation

Get started with LosenskySystems Web Components by installing the package and setting up your project.

## Package Installation

Install the core package using your preferred package manager:

### npm
```bash
npm install @losensky-systems/web-components-core
```

### pnpm
```bash
pnpm add @losensky-systems/web-components-core
```

### yarn
```bash
yarn add @losensky-systems/web-components-core
```

## CSS Setup

Import the CSS styles in your main CSS file or at the root of your application:

```css
@import '@losensky-systems/web-components-css';
```

Or if you're using a bundler that supports CSS imports:

```javascript
import '@losensky-systems/web-components-css'
```

## Basic Usage

Once installed, you can import and use components in your React application:

```tsx
import { Button, Card, Input } from '@losensky-systems/web-components-core'

function App() {
  return (
    <div>
      <Card>
        <Card.Header>
          <h2>Welcome</h2>
        </Card.Header>
        <Card.Body>
          <Input placeholder="Enter your name" />
          <Button>Submit</Button>
        </Card.Body>
      </Card>
    </div>
  )
}
```

## TypeScript Support

The package includes full TypeScript support with type definitions. No additional setup is required.

## Framework Compatibility

LosenskySystems Web Components are built with React and work with:

- **React 18+**
- **Next.js**
- **Vite**
- **Create React App**
- **Any React-based framework**
