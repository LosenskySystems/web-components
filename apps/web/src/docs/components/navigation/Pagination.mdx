# Pagination Component

The Pagination component provides navigation through multiple pages of data with page numbers, prev/next buttons, and optional first/last controls.

## Import

```tsx
import { Pagination } from '@losensky-systems/web-components-core'
```

## Basic Usage

<CodeBlock code={`function PaginationExample() {
  const [currentPage, setCurrentPage] = useState(1)
  const totalPages = 10

  return (
    <Pagination
      currentPage={currentPage}
      totalPages={totalPages}
      onPageChange={setCurrentPage}
    />
  )
}`}>
  <PaginationBasicExample />
</CodeBlock>
