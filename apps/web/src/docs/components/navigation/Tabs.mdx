# Tabs

A tabs component for organizing content into separate views that can be switched between.

## Basic Usage

<CodeBlock code={`<Tabs defaultValue="tab1">
  <TabsList>
    <TabsTrigger value="tab1">Tab 1</TabsTrigger>
    <TabsTrigger value="tab2">Tab 2</TabsTrigger>
    <TabsTrigger value="tab3">Tab 3</TabsTrigger>
  </TabsList>
  <TabsContent value="tab1">
    <div className="p-4">
      <h3 className="text-lg font-semibold mb-2">Tab 1 Content</h3>
      <p>This is the content for the first tab.</p>
    </div>
  </TabsContent>
  <TabsContent value="tab2">
    <div className="p-4">
      <h3 className="text-lg font-semibold mb-2">Tab 2 Content</h3>

    </div>
  </TabsContent>
  <TabsContent value="tab3">
    <div className="p-4">
      <h3 className="text-lg font-semibold mb-2">Tab 3 Content</h3>
      <p>This is the content for the third tab.</p>
    </div>
  </TabsContent>
</Tabs>`}>

## Props

### Tabs Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `defaultValue` | `string` | - | Default active tab value |
| `value` | `string` | - | Controlled active tab value |
| `onValueChange` | `(value: string) => void` | - | Callback when active tab changes |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Tabs size |
| `variant` | `'default' \| 'pills' \| 'underline'` | `'default'` | Tabs variant |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Tabs orientation |
| `className` | `string` | - | Additional CSS classes |
| `children` | `React.ReactNode` | - | Tab children |

### TabsTrigger Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Tab value |
| `disabled` | `boolean` | `false` | Whether the tab is disabled |
| `className` | `string` | - | Additional CSS classes |
| `children` | `React.ReactNode` | - | Tab content |
| `icon` | `React.ReactNode` | - | Icon to display before content |

### TabsContent Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Tab value |
| `className` | `string` | - | Additional CSS classes |
| `children` | `React.ReactNode` | - | Content children |
| `forceMount` | `boolean` | `false` | Whether to force mount the content |
