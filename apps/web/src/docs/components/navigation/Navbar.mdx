# Navbar

A flexible navigation bar component with compound architecture for building responsive headers with brand, navigation links, and action buttons.

## Basic Usage

<CodeBlock code={`<Navbar>
  <Navbar.Brand href="/">MyApp</Navbar.Brand>
  <Navbar.Nav>
    <Navbar.Item href="/" active>Home</Navbar.Item>
    <Navbar.Item href="/about">About</Navbar.Item>
    <Navbar.Item href="/services">Services</Navbar.Item>
    <Navbar.Item href="/contact">Contact</Navbar.Item>
  </Navbar.Nav>
</Navbar>`}>
  <Navbar>
    <Navbar.Brand href="/">MyApp</Navbar.Brand>
    <Navbar.Nav>
      <Navbar.Item href="/" active>Home</Navbar.Item>
      <Navbar.Item href="/about">About</Navbar.Item>
      <Navbar.Item href="/services">Services</Navbar.Item>
      <Navbar.Item href="/contact">Contact</Navbar.Item>
    </Navbar.Nav>
  </Navbar>
</CodeBlock>

## With Mobile Toggle

<CodeBlock code={`function MobileNavbar() {
  const [isOpen, setIsOpen] = React.useState(false);

  return (
    <Navbar bordered shadow>
      <Navbar.Brand href="/">MyApp</Navbar.Brand>
      <Navbar.Toggle 
        onClick={() => setIsOpen(!isOpen)} 
        isOpen={isOpen} 
      />
      <Navbar.Nav>
        <Navbar.Item href="/" active>Home</Navbar.Item>
        <Navbar.Item href="/about">About</Navbar.Item>
        <Navbar.Item href="/services">Services</Navbar.Item>
      </Navbar.Nav>
      <Navbar.Actions>
        <Button size="sm">Sign In</Button>
      </Navbar.Actions>
    </Navbar>
  );
}`}>
  <div className="bg-gray-50 p-4 rounded">
    <p className="text-sm text-gray-600 mb-4">Toggle button appears on mobile. Navigation items are hidden on mobile by default.</p>
    <Navbar bordered shadow>
      <Navbar.Brand href="/">MyApp</Navbar.Brand>
      <Navbar.Toggle onClick={() => {}} />
      <Navbar.Nav>
        <Navbar.Item href="/" active>Home</Navbar.Item>
        <Navbar.Item href="/about">About</Navbar.Item>
        <Navbar.Item href="/services">Services</Navbar.Item>
      </Navbar.Nav>
      <Navbar.Actions>
        <Button size="sm">Sign In</Button>
      </Navbar.Actions>
    </Navbar>
  </div>
</CodeBlock>
