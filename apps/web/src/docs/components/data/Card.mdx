import { Card, Button } from '@losensky-systems/web-components-core'

# Card Component

The Card component provides a flexible container with compound component architecture for building content layouts. It supports various variants, sizes, and interactive states.

## Import

```tsx
import { Card } from '@losensky-systems/web-components-core'
```

## Basic Usage

<div style={{ marginBottom: '2rem' }}>
  <Card>
    <Card.Body>
      <p>This is a basic card with some content.</p>
    </Card.Body>
  </Card>
</div>

```tsx
<Card>
  <Card.Body>
    <p>This is a basic card with some content.</p>
  </Card.Body>
</Card>
```

## Props

### Card Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `variant` | `'default' \| 'outlined' \| 'elevated' \| 'filled'` | `'default'` | Card visual variant |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Card size |
| `padding` | `'none' \| 'sm' \| 'md' \| 'lg'` | `'md'` | Card padding |
| `rounded` | `'none' \| 'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Border radius |
| `hover` | `boolean` | `false` | Enable hover effect |
| `clickable` | `boolean` | `false` | Make card clickable |
| `onClick` | `(event: React.MouseEvent<HTMLDivElement>) => void` | - | Click handler |
| `className` | `string` | `''` | Additional CSS classes |

### Card.Header Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `title` | `string` | - | Header title |
| `subtitle` | `string` | - | Header subtitle |
| `avatar` | `React.ReactNode` | - | Avatar element |
| `action` | `React.ReactNode` | - | Action element |
| `className` | `string` | `''` | Additional CSS classes |

### Card.Body Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `''` | Additional CSS classes |

### Card.Footer Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `className` | `string` | `''` | Additional CSS classes |

### Card.Actions Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `align` | `'left' \| 'center' \| 'right' \| 'between'` | `'right'` | Action alignment |
| `spacing` | `'none' \| 'sm' \| 'md' \| 'lg'` | `'md'` | Action spacing |
| `className` | `string` | `''` | Additional CSS classes |
