import { EmptyState, Button, Icon } from '@losensky-systems/web-components-core'

# Empty State Component

The Empty State component provides a centered placeholder for displaying no data or empty content states. It supports icons, titles, descriptions, and action buttons.

## Import

```tsx
import { EmptyState } from '@losensky-systems/web-components-core'
```

## Basic Usage

<CodeBlock code={`<EmptyState 
  title="No items found"
  description="There are no items to display at this time"
/>`}>
  <EmptyState 
    title="No items found"
    description="There are no items to display at this time"
  />
</CodeBlock>

## With Icon

<CodeBlock code={`<EmptyState 
  icon={<Icon size="xl" />}
  title="No results"
  description="We couldn't find anything matching your search"
/>`}>
  <EmptyState 
    icon={<Icon size="xl" />}
    title="No results"
    description="We couldn't find anything matching your search"
  />
</CodeBlock>

## With Action

<CodeBlock code={`<EmptyState 
  title="No data available"
  description="Get started by creating your first item"
  action={<Button>Create Item</Button>}
/>`}>
  <EmptyState 
    title="No data available"
    description="Get started by creating your first item"
    action={<Button>Create Item</Button>}
  />
</CodeBlock>
