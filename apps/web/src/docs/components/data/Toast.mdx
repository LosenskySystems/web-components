# Toast

A floating notification component for displaying temporary messages and feedback.

## Basic Usage

<CodeBlock code={`<Toast title="Success!" description="Your action was completed successfully." />

// Or programmatically
const showToast = () => {
  const toast = document.createElement('div');
  toast.innerHTML = '<div class="web-toast web-toast-success web-toast-top-right"><div class="web-toast-icon">‚úì</div><div class="web-toast-content"><div class="web-toast-title">Success!</div><div class="web-toast-description">Your action was completed successfully.</div></div></div>';
  toast.style.position = 'fixed';
  toast.style.top = '20px';
  toast.style.right = '20px';
  toast.style.zIndex = '9999';
  document.body.appendChild(toast);
  setTimeout(() => document.body.removeChild(toast), 5000);
};`}>
  <div className="space-y-4">
    <Toast title="Success!" description="Your action was completed successfully." />
    
    <Button onClick={() => {
      const toast = document.createElement('div');
      toast.innerHTML = '<div class="web-toast web-toast-success web-toast-top-right"><div class="web-toast-icon">‚úì</div><div class="web-toast-content"><div class="web-toast-title">Success!</div><div class="web-toast-description">Your action was completed successfully.</div></div></div>';
      toast.style.position = 'fixed';
      toast.style.top = '20px';
      toast.style.right = '20px';
      toast.style.zIndex = '9999';
      document.body.appendChild(toast);
      setTimeout(() => document.body.removeChild(toast), 5000);
    }}>Show Basic Toast</Button>
  </div>
</CodeBlock>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Custom content |
| `title` | `string` | - | Toast title |
| `description` | `string` | - | Toast description |
| `variant` | `'default' \| 'success' \| 'warning' \| 'error' \| 'info'` | `'default'` | Toast variant |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Toast size |
| `position` | `'top-left' \| 'top-right' \| 'top-center' \| 'bottom-left' \| 'bottom-right' \| 'bottom-center'` | `'top-right'` | Toast position |
| `closable` | `boolean` | `true` | Whether to show close button |
| `duration` | `number` | `5000` | Auto dismiss duration in milliseconds (0 = no auto dismiss) |
| `onClose` | `() => void` | - | Callback when toast is closed |
| `icon` | `React.ReactNode` | - | Custom icon |
| `className` | `string` | - | Additional CSS classes |
