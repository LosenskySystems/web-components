import { Callout, Button, Icon } from '@losensky-systems/web-components-core'

# Callout Component

The Callout component is designed for emphasizing important information inline. It features a left border, optional icons, and variant support for different message types.

## Import

```tsx
import { Callout } from '@losensky-systems/web-components-core'
```

## Basic Usage

<CodeBlock code={`<Callout description="This is important information you should know." />`}>
  <Callout description="This is important information you should know." />
</CodeBlock>

## With Title

<CodeBlock code={`<Callout 
  title="Note"
  description="This is an important note about the feature."
/>`}>
  <Callout 
    title="Note"
    description="This is an important note about the feature."
  />
</CodeBlock>

## With Icon

<CodeBlock code={`<Callout 
  variant="info"
  title="Documentation"
  icon={<Icon size="md" />}
  description="You can find more details in our documentation."
/>`}>
  <Callout 
    variant="info"
    title="Documentation"
    icon={<Icon size="md" />}
    description="You can find more details in our documentation."
  />
</CodeBlock>

## With Action

<CodeBlock code={`<Callout 
  variant="warning"
  title="Update Available"
  description="A new version is available with improved features."
  action={<Button size="sm">Update Now</Button>}
/>`}>
  <Callout 
    variant="warning"
    title="Update Available"
    description="A new version is available with improved features."
    action={<Button size="sm">Update Now</Button>}
  />
</CodeBlock>
