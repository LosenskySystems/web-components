# List Component

The List component provides a flexible container for displaying collections of items with support for avatars, icons, descriptions, actions, and interactive states. Perfect for contact lists, menus, and data display.

## Import

```tsx
import { List } from '@losensky-systems/web-components-core'
```

## Basic Usage

<CodeBlock code={`<List>
  <List.Item title="Email notifications" />
  <List.Item title="SMS notifications" />
  <List.Item title="Push notifications" />
</List>`}>
  <div style={{ marginBottom: '2rem' }}>
    <List>
      <List.Item title="Email notifications" />
      <List.Item title="SMS notifications" />
      <List.Item title="Push notifications" />
    </List>
  </div>
</CodeBlock>

## Accessibility

The List component includes comprehensive accessibility features:

- **Semantic HTML**: Uses proper `<ul>` and `<li>` elements
- **Keyboard navigation**: Full keyboard support for interactive items (Enter and Space keys)
- **ARIA attributes**: Proper `role` and `aria-disabled` attributes for interactive and disabled states
- **Focus management**: Proper focus handling and visual indicators
- **Screen reader friendly**: Clear structure for screen readers
