# Badge Component

The Badge component displays a notification count or indicator that overlays any element. Perfect for shopping carts, notifications, messages, and more.

## Import

```tsx
import { Badge } from '@losensky-systems/web-components-core'
```

## Basic Usage

Wrap any element with a Badge to show a count or text:

<CodeBlock code={`<Badge count={2}>
  <Button icon={<ShoppingCart size={16} />} iconOnly variant="ghost" />
</Badge>

<Badge count={5}>
  <Button icon={<Bell size={16} />} iconOnly variant="ghost" />
</Badge>

<Badge count={12}>
  <Button icon={<MessageCircle size={16} />} iconOnly variant="ghost" />
</Badge>

<Badge count={0} showZero>
  <Button icon={<Heart size={16} />} iconOnly variant="ghost" />
</Badge>

<Badge text="New">
  <Button size="sm">Product</Button>
</Badge>

<Badge text="Sale" variant="error">
  <Button size="sm" variant="outline">Item</Button>
</Badge>`}>
  <div className="flex gap-4 mb-8 flex-wrap items-center">
    <Badge count={2}>
      <Button icon={<ShoppingCart size={16} />} iconOnly variant="ghost" />
    </Badge>
    
    <Badge count={5}>
      <Button icon={<Bell size={16} />} iconOnly variant="ghost" />
    </Badge>
    
    <Badge count={12}>
      <Button icon={<MessageCircle size={16} />} iconOnly variant="ghost" />
    </Badge>
    
    <Badge count={0} showZero>
      <Button icon={<Heart size={16} />} iconOnly variant="ghost" />
    </Badge>
    
    <Badge text="New">
      <Button size="sm">Product</Button>
    </Badge>
    
    <Badge text="Sale" variant="error">
      <Button size="sm" variant="outline">Item</Button>
    </Badge>
  </div>
</CodeBlock>

## Accessibility

The Badge component includes proper accessibility features:

- **ARIA attributes**: `role="status"` and `aria-label`
- **Screen reader friendly**: Announces count or notification status
- **Keyboard accessible**: Works with focusable elements
- **Semantic meaning**: Clear indication of notification state
