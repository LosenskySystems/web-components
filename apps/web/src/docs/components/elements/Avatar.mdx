# Avatar

A simple avatar component that displays user profile images with fallback support.

## Basic Usage

<CodeBlock code={`<Avatar src="https://images.unsplash.com/photo-1472099645785-5658abf4ff4e?w=100&h=100&fit=crop&crop=face" alt="John Doe" />`}>
  <Avatar src="https://images.unsplash.com/photo-1472099645785-5658abf4ff4e?w=100&h=100&fit=crop&crop=face" alt="John Doe" />
</CodeBlock>

## With Fallback Text

<CodeBlock code={`<Avatar fallback="JD" alt="John Doe" />`}>
  <Avatar fallback="JD" alt="John Doe" />
</CodeBlock>
