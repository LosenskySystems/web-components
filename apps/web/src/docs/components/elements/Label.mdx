import { Label } from '@losensky-systems/web-components-core'

# Label Component

The Label component provides accessible labels for form elements with proper contrast and styling.

## Import

```tsx
import { Label } from '@losensky-systems/web-components-core'
```

## Basic Usage

<CodeBlock code={`<Label htmlFor="email">Email Address</Label>
<Label htmlFor="password">Password</Label>
<Label htmlFor="name" required>Full Name</Label>`}>
  <div className="mb-8">
    <Label htmlFor="email">Email Address</Label>
    <Label htmlFor="password">Password</Label>
    <Label htmlFor="name" required>Full Name</Label>
  </div>
</CodeBlock>

## Sizes

<CodeBlock code={`<Label size="sm" htmlFor="small">Small Label</Label>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Label text content |
| `htmlFor` | `string` | - | ID of associated form element |
| `required` | `boolean` | `false` | Show required asterisk |
| `disabled` | `boolean` | `false` | Disabled state |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Label size |
| `color` | `'default' \| 'muted' \| 'primary' \| 'success' \| 'warning' \| 'error' \| 'info'` | `'default'` | Label color |
| `helperText` | `string` | - | Helper text below label |
| `errorText` | `string` | - | Error text below label |
| `successText` | `string` | - | Success text below label |
| `className` | `string` | `''` | Additional CSS classes |
