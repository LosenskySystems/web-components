# Button Component

The Button component is a complete, accessible button with built-in styles. Use Tailwind classes for easy customization of colors, borders, and other details.

## Import

```tsx
import { Button } from '@losensky-systems/web-components-core'
```

## Variants

The Button component supports multiple predefined variants:

<div style={{ display: 'flex', gap: '1rem', marginBottom: '2rem', flexWrap: 'wrap' }}>
  <Button variant="primary">Primary</Button>
  <Button variant="secondary">Secondary</Button>
  <Button variant="outline">Outline</Button>
  <Button variant="ghost">Ghost</Button>
  <Button variant="destructive">Destructive</Button>
</div>

## Loading State

The Button component supports a loading state with proper accessibility:

<div style={{ display: 'flex', gap: '1rem', marginBottom: '2rem', flexWrap: 'wrap' }}>
  <Button loading>
    Loading Button
  </Button>
  <Button loading loadingText="Saving data...">
    Custom Loading Text
  </Button>
  <Button variant="outline" loading>
    Outline Loading
  </Button>
</div>

```tsx
<Button loading>
  Loading Button
</Button>

<Button loading loadingText="Saving data...">
  Custom Loading Text
</Button>

<Button variant="outline" loading>
  Outline Loading
</Button>
```

## Customization with Tailwind

You can easily add Tailwind classes to any button variant to customize colors, borders, and other details:

<div style={{ display: 'flex', gap: '1rem', marginBottom: '2rem', flexWrap: 'wrap' }}>
  <Button variant="primary" className="bg-violet-800 text-white">
    Violet Primary
  </Button>
  <Button variant="secondary" className="bg-green-100 text-green-800 border-green-300">
    Green Secondary
  </Button>
  <Button variant="outline" className="border-purple-500 text-purple-500">
    Purple Outline
  </Button>
</div>

```tsx
<Button variant="primary" className="bg-violet-800 text-white">
  Violet Primary
</Button>

<Button variant="secondary" className="bg-green-100 text-green-800 border-green-300">
  Green Secondary
</Button>

<Button variant="outline" className="border-purple-500 text-purple-500">
  Purple Outline
</Button>
```

**Note:** Simply add Tailwind classes to override specific properties. The button keeps all its base styling (shadows, hover effects, etc.) and only the specified properties are changed.
