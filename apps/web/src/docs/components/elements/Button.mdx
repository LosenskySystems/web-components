# Button Component

The Button component is a complete, accessible button with built-in styles. Use Tailwind classes for easy customization of colors, borders, and other details.

## Import

```tsx
import { Button, Tooltip } from '@losensky-systems/web-components-core'
import { FolderOpen, Save, Plus, Settings, Heart } from 'lucide-react'
```

### Icon Props

- **`icon`**: React node (SVG, icon component, emoji)
- **`iconPosition`**: `'left'` | `'right'` (default: `'left'`)
- **`iconOnly`**: `boolean` - Shows only the icon with screen reader text

### Using Lucide React Icons

```tsx
import { Download, Search, Heart } from 'lucide-react'

<Button icon={<Download size={16} />} iconPosition="right">
  Download
</Button>

<Button icon={<Search size={16} />} iconOnly variant="ghost">
  Search
</Button>

<Button icon={<Heart size={16} />} iconOnly color="error">
  Like
</Button>
```

### Icon Accessibility

- Icons are marked with `aria-hidden="true"` to prevent screen reader confusion
- Icon-only buttons use `aria-label` for accessibility
- Text content is preserved in `sr-only` span for icon-only buttons

## Button Groups

Use `Button.Group` to group related buttons together:

<div style={{ display: 'flex', gap: '2rem', marginBottom: '2rem', flexWrap: 'wrap', alignItems: 'flex-start' }}>
  <div>
    <h4 style={{ margin: '0 0 0.5rem 0', fontSize: '0.875rem', fontWeight: '600' }}>Basic Group</h4>
    <Button.Group>
      <Button variant="primary">Save</Button>
      <Button variant="secondary">Cancel</Button>
    </Button.Group>
  </div>
  
  <div>
    <h4 style={{ margin: '0 0 0.5rem 0', fontSize: '0.875rem', fontWeight: '600' }}>Attached</h4>
    <Button.Group attached>
      <Button variant="primary">Save</Button>
      <Button variant="secondary">Cancel</Button>
    </Button.Group>
  </div>
  
  <div>
    <h4 style={{ margin: '0 0 0.5rem 0', fontSize: '0.875rem', fontWeight: '600' }}>With Icons</h4>
    <Button.Group>
      <Button icon="📁" iconPosition="left">Open</Button>
      <Button icon="💾" iconPosition="left">Save</Button>
      <Button icon="🗑" iconPosition="left" variant="destructive">Delete</Button>
    </Button.Group>
  </div>
</div>

```tsx
// Basic button group
<Button.Group>
  <Button variant="primary">Save</Button>
  <Button variant="secondary">Cancel</Button>
</Button.Group>

// Attached buttons (connected borders)
<Button.Group attached>
  <Button variant="primary">Save</Button>
  <Button variant="secondary">Cancel</Button>
</Button.Group>

// With icons
<Button.Group>
  <Button icon="📁" iconPosition="left">Open</Button>
  <Button icon="💾" iconPosition="left">Save</Button>
  <Button icon="🗑" iconPosition="left" variant="destructive">Delete</Button>
</Button.Group>
```

### Group Props

- **`orientation`**: `'horizontal'` | `'vertical'` (default: `'horizontal'`)
- **`spacing`**: `'none'` | `'sm'` | `'md'` | `'lg'` (default: `'sm'`)
- **`attached`**: `boolean` - Connects buttons with shared borders

## Customization with Tailwind

You can easily add Tailwind classes to any button variant to customize colors, borders, and other details:

<div style={{ display: 'flex', gap: '1rem', marginBottom: '2rem', flexWrap: 'wrap' }}>
  <Button variant="primary" className="bg-violet-800 text-white">
    Violet Primary
  </Button>
  <Button variant="secondary" className="bg-green-100 text-green-800 border-green-300">
    Green Secondary
  </Button>
  <Button variant="outline" className="border-purple-500 text-purple-500">
    Purple Outline
  </Button>
</div>

```tsx
<Button variant="primary" className="bg-violet-800 text-white">
  Violet Primary
</Button>

<Button variant="secondary" className="bg-green-100 text-green-800 border-green-300">
  Green Secondary
</Button>

<Button variant="outline" className="border-purple-500 text-purple-500">
  Purple Outline
</Button>
```

**Note:** Simply add Tailwind classes to override specific properties. The button keeps all its base styling (shadows, hover effects, etc.) and only the specified properties are changed.

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Button content |
| `variant` | `'primary' \| 'secondary' \| 'outline' \| 'ghost' \| 'text' \| 'destructive' \| 'custom'` | `'primary'` | Button style variant |
| `color` | `'default' \| 'success' \| 'info' \| 'warning' \| 'error'` | `'default'` | Semantic color theme |
| `size` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Button size |
| `onClick` | `(event: React.MouseEvent<HTMLButtonElement>) => void` | - | Click handler |
| `className` | `string` | `''` | Additional CSS classes |
| `disabled` | `boolean` | `false` | Disabled state |
| `type` | `'button' \| 'submit' \| 'reset'` | `'button'` | Button type |
| `loading` | `boolean` | `false` | Loading state |
| `loadingText` | `string` | `'Loading...'` | Loading text |
| `icon` | `React.ReactNode` | - | Icon element |
| `iconPosition` | `'left' \| 'right'` | `'left'` | Icon position |
| `iconOnly` | `boolean` | `false` | Show only icon |
