# Tooltip Component

The Tooltip component provides contextual information on hover, click, or focus. It wraps any element and shows helpful text or content.

## Import

```tsx
import { Tooltip } from '@losensky-systems/web-components-core'
```

## Basic Usage

Wrap any element with the Tooltip component:

<div style={{ display: 'flex', gap: '1rem', marginBottom: '2rem', flexWrap: 'wrap', alignItems: 'center' }}>
  <Tooltip content="This is a helpful tooltip">
    <Button>Hover me</Button>
  </Tooltip>
  
  <Tooltip content="Click to see tooltip" trigger="click">
    <Button variant="outline">Click me</Button>
  </Tooltip>
  
  <Tooltip content="Focus to see tooltip" trigger="focus">
    <Button variant="ghost">Focus me</Button>
  </Tooltip>
</div>

```tsx

```

## Triggers

Choose how the tooltip is activated:

### Hover (Default)
Shows on mouse enter, hides on mouse leave:

```tsx
<Tooltip content="Hover tooltip">
  <Button>Hover me</Button>
</Tooltip>
```

### Click
Shows/hides on click, closes when clicking outside:

```tsx
<Tooltip content="Click tooltip" trigger="click">
  <Button>Click me</Button>
</Tooltip>
```

### Focus
Shows on focus, hides on blur (useful for keyboard navigation):

```tsx
<Tooltip content="Focus tooltip" trigger="focus">
  <Button>Focus me</Button>
</Tooltip>
```

## Accessibility

The Tooltip component includes proper accessibility features:

- **ARIA attributes**: `role="tooltip"` and `aria-hidden`
- **Keyboard support**: Works with focus triggers
- **Screen reader friendly**: Content is properly announced
- **Focus management**: Handles focus/blur events correctly

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Element to wrap with tooltip |
| `content` | `React.ReactNode` | - | Tooltip content |
| `placement` | `'top' \| 'bottom' \| 'left' \| 'right'` | `'top'` | Tooltip position |
| `trigger` | `'hover' \| 'click' \| 'focus'` | `'hover'` | Activation method |
| `delay` | `number` | `200` | Delay before showing (ms) |
| `className` | `string` | `''` | Additional CSS classes |
| `disabled` | `boolean` | `false` | Disable tooltip |
