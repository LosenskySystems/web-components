# Dropdown

A simple dropdown menu component for displaying a list of actions or options.

## Basic Usage

<CodeBlock code={`<Dropdown 
  items={[
    { children: 'Edit', onClick: () => console.log('Edit clicked') },
    { children: 'Delete', onClick: () => console.log('Delete clicked') },
    { children: 'Share', onClick: () => console.log('Share clicked') }
  ]}
>
  <Button>Actions</Button>
</Dropdown>`}>
  <Dropdown 
    items={[
      { children: 'Edit', onClick: () => console.log('Edit clicked') },
      { children: 'Delete', onClick: () => console.log('Delete clicked') },
      { children: 'Share', onClick: () => console.log('Share clicked') }
    ]}
  >
    <Button>Actions</Button>
  </Dropdown>
</CodeBlock>

## Props

### Dropdown Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | The trigger element |
| `items` | `DropdownItemProps[]` | - | Array of dropdown items |
| `placement` | `'top' \| 'bottom' \| 'left' \| 'right'` | `'bottom'` | Placement of the dropdown |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Size variant |
| `isOpen` | `boolean` | - | Whether the dropdown is open (controlled) |
| `onOpenChange` | `(isOpen: boolean) => void` | - | Callback when dropdown open state changes |
| `className` | `string` | - | Additional CSS classes |

### DropdownItem Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `children` | `React.ReactNode` | - | Item content |
| `icon` | `React.ReactNode` | - | Icon to display before the text |
| `disabled` | `boolean` | `false` | Whether the item is disabled |
| `onClick` | `() => void` | - | Click handler |
| `className` | `string` | - | Additional CSS classes |
