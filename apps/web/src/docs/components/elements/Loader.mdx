# Loader Component

The Loader component displays loading states with customizable spinners, text, or both.

## Import

```tsx
import { Loader } from '@losensky-systems/web-components-core'
```

## Basic Usage

<CodeBlock code={`<Loader />
<Loader type="text" />
<Loader type="both" />`}>
  <div className="flex gap-8 mb-8 items-center">
    <Loader />
    <Loader type="text" />
    <Loader type="both" />
  </div>
</CodeBlock>

## Sizes

Loaders come in three sizes:

<CodeBlock code={`<Loader size="sm" />
<Loader size="md" />
<Loader size="lg" />`}>
  <div className="flex gap-8 mb-8 items-center">
    <Loader size="sm" />
    <Loader size="md" />
    <Loader size="lg" />
  </div>
</CodeBlock>

## Variants

Different color variants for different contexts:

<CodeBlock code={`<Loader variant="primary" />
<Loader variant="secondary" />
<Loader variant="white" />`}>
  <div className="flex gap-8 mb-8 items-center">
    <Loader variant="primary" />
    <Loader variant="secondary" />
    <div className="bg-gray-800 p-4 rounded-lg">
      <Loader variant="white" />
    </div>
  </div>
</CodeBlock>

## Custom Loading Text

Customize the loading message:

<CodeBlock code={`<Loader type="text" text="Processing..." />
<Loader type="both" text="Saving data..." />
<Loader type="text" text="Please wait..." />`}>
  <div className="flex gap-8 mb-8 items-center">
    <Loader type="text" text="Processing..." />
    <Loader type="both" text="Saving data..." />
    <Loader type="text" text="Please wait..." />
  </div>
</CodeBlock>

## In Context

Loaders work great within buttons and other components:

<CodeBlock code={`// In a button
<button className="px-4 py-2 bg-blue-600 text-white border-none rounded-md flex items-center gap-2">
  <Loader size="sm" variant="white" type="spinner" />
  Loading...
</button>

// In a card
<div className="p-4 border border-gray-200 rounded-lg flex items-center gap-4">
  <Loader size="md" />
  <span>Processing your request</span>
</div>`}>
  <div className="flex gap-4 mb-8 flex-wrap">
    <button className="px-4 py-2 bg-blue-600 text-white border-none rounded-md flex items-center gap-2">
      <Loader size="sm" variant="white" type="spinner" />
      Loading...
    </button>
    <div className="p-4 border border-gray-200 rounded-lg flex items-center gap-4">
      <Loader size="md" />
      <span>Processing your request</span>
    </div>
  </div>
</CodeBlock>

## Customization

You can customize loaders using Tailwind classes:

<CodeBlock code={`<Loader className="text-red-500" />
<Loader className="text-green-500" type="text" text="Custom Color" />`}>
  <div className="flex gap-8 mb-8 items-center">
    <Loader className="text-red-500" />
    <Loader className="text-green-500" type="text" text="Custom Color" />
  </div>
</CodeBlock>
