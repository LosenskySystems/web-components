# Chip Component

The Chip component displays compact labels or tags, perfect for filters, categories, removable items, and status indicators.

## Import

```tsx
import { Chip } from '@losensky-systems/web-components-core'
```

## Basic Usage

Simple chips for labels and tags:

<CodeBlock code={`<Chip>Label</Chip>
<Chip variant="success">Success</Chip>
<Chip variant="warning">Warning</Chip>
<Chip variant="error">Error</Chip>
<Chip variant="info">Info</Chip>`}>
  <div className="flex gap-4 mb-8 flex-wrap">
    <Chip>Label</Chip>
    <Chip variant="success">Success</Chip>
    <Chip variant="warning">Warning</Chip>
    <Chip variant="error">Error</Chip>
    <Chip variant="info">Info</Chip>
  </div>
</CodeBlock>

## Sizes

Three size variants for different contexts:

<CodeBlock code={`<Chip size="sm">Small Chip</Chip>
<Chip size="md">Medium Chip</Chip>
<Chip size="lg">Large Chip</Chip>`}>
  <div className="flex gap-4 mb-8 flex-wrap items-center">
    <Chip size="sm">Small Chip</Chip>
    <Chip size="md">Medium Chip</Chip>
    <Chip size="lg">Large Chip</Chip>
  </div>
</CodeBlock>

## Removable Chips

Default behavior includes a remove button:

<CodeBlock code={`<Chip removable onRemove={() => alert('Removed!')}>Removable</Chip>
<Chip removable variant="success" onRemove={() => alert('Removed!')}>Success</Chip>
<Chip removable variant="error" onRemove={() => alert('Removed!')}>Error</Chip>`}>
  <div className="flex gap-4 mb-8 flex-wrap">
    <Chip removable onRemove={() => alert('Removed!')}>Removable</Chip>
    <Chip removable variant="success" onRemove={() => alert('Removed!')}>Success</Chip>
    <Chip removable variant="error" onRemove={() => alert('Removed!')}>Error</Chip>
  </div>
</CodeBlock>

## Non-Removable Chips

Disable the remove button for static labels:

<CodeBlock code={`<Chip removable={false}>Always Visible</Chip>
<Chip removable={false} variant="success">Completed</Chip>
<Chip removable={false} variant="warning">Pending</Chip>`}>
  <div className="flex gap-4 mb-8 flex-wrap">
    <Chip removable={false}>Always Visible</Chip>
    <Chip removable={false} variant="success">Completed</Chip>
    <Chip removable={false} variant="warning">Pending</Chip>
  </div>
</CodeBlock>

## With Icons

Add icons to chips using the leftSlot:

<CodeBlock code={`<Chip leftSlot={<CheckCircle size={16} />}>Verified</Chip>
<Chip leftSlot={<XCircle size={16} />} variant="error">Failed</Chip>
<Chip leftSlot={<AlertCircle size={16} />} variant="warning">Warning</Chip>
<Chip leftSlot={<Info size={16} />} variant="info">Information</Chip>`}>
  <div className="flex gap-4 mb-8 flex-wrap">
    <Chip leftSlot={<CheckCircle size={16} />}>Verified</Chip>
    <Chip leftSlot={<XCircle size={16} />} variant="error">Failed</Chip>
    <Chip leftSlot={<AlertCircle size={16} />} variant="warning">Warning</Chip>
    <Chip leftSlot={<Info size={16} />} variant="info">Information</Chip>
  </div>
</CodeBlock>

Display avatars with chips for user tags:

## Disabled State

Disabled chips cannot be removed:

<CodeBlock code={`<Chip disabled>Disabled Chip</Chip>
<Chip disabled variant="success">Disabled Success</Chip>
<Chip disabled removable onRemove={() => {}}>Cannot Remove</Chip>`}>
  <div className="flex gap-4 mb-8 flex-wrap">
    <Chip disabled>Disabled Chip</Chip>
    <Chip disabled variant="success">Disabled Success</Chip>
    <Chip disabled removable onRemove={() => {}}>Cannot Remove</Chip>
  </div>
</CodeBlock>
