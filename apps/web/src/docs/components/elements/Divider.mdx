import { Divider } from '@losensky-systems/web-components-core'

# Divider Component

The Divider component creates visual separation between content sections. It supports horizontal and vertical orientations, different styles, and can include labels.

## Import

```tsx
import { Divider } from '@losensky-systems/web-components-core'
```

## Basic Usage

Simple dividers for content separation:

<div style={{ marginBottom: '2rem' }}>
  <p>Content above the divider</p>
  <Divider />
  <p>Content below the divider</p>
</div>

```tsx
<p>Content above the divider</p>
<Divider />
<p>Content below the divider</p>
```

## Orientations

### Horizontal (Default)
Horizontal dividers separate content vertically:

<div style={{ marginBottom: '2rem' }}>
  <div style={{ padding: '1rem', background: '#f8fafc', borderRadius: '0.5rem', marginBottom: '1rem' }}>
    Section 1
  </div>
  <Divider />
  <div style={{ padding: '1rem', background: '#f8fafc', borderRadius: '0.5rem', marginTop: '1rem' }}>
    Section 2
  </div>
</div>

```tsx
<div>Section 1</div>
<Divider />
<div>Section 2</div>
```

### Vertical
Vertical dividers separate content horizontally:

<div style={{ display: 'flex', alignItems: 'center', marginBottom: '2rem', height: '4rem' }}>
  <div style={{ padding: '0.5rem 1rem', background: '#f8fafc', borderRadius: '0.5rem' }}>
    Left Content
  </div>
  <Divider orientation="vertical" />
  <div style={{ padding: '0.5rem 1rem', background: '#f8fafc', borderRadius: '0.5rem' }}>
    Right Content
  </div>
</div>

```tsx
<div style={{ display: 'flex', alignItems: 'center' }}>
  <div>Left Content</div>
  <Divider orientation="vertical" />
  <div>Right Content</div>
</div>
```

## Variants

Different border styles for visual variety:

<div style={{ marginBottom: '2rem' }}>
  <p>Solid divider (default):</p>
  <Divider variant="solid" />
  
  <p style={{ marginTop: '1rem' }}>Dashed divider:</p>
  <Divider variant="dashed" />
  
  <p style={{ marginTop: '1rem' }}>Dotted divider:</p>
  <Divider variant="dotted" />
</div>

```tsx
<Divider variant="solid" />   {/* Default */}
<Divider variant="dashed" />
<Divider variant="dotted" />
```

## Spacing

Control the margin around dividers:

<div style={{ marginBottom: '2rem' }}>
  <p>Small spacing:</p>
  <Divider spacing="sm" />
  <p>Medium spacing (default):</p>
  <Divider spacing="md" />
  <p>Large spacing:</p>
  <Divider spacing="lg" />
  <p>Extra large spacing:</p>
  <Divider spacing="xl" />
</div>

```tsx
<Divider spacing="sm" />   {/* 0.25rem margin */}
<Divider spacing="md" />    {/* 0.5rem margin - default */}
<Divider spacing="lg" />   {/* 1rem margin */}
<Divider spacing="xl" />   {/* 1.5rem margin */}
```

## Accessibility

The Divider component includes proper accessibility features:

- **ARIA attributes**: `role="separator"` and `aria-orientation`
- **Semantic meaning**: Clear indication of content separation
- **Screen reader friendly**: Proper announcements for content structure
- **Keyboard accessible**: Works with focus management

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Divider direction |
| `variant` | `'solid' \| 'dashed' \| 'dotted'` | `'solid'` | Border style |
| `spacing` | `'sm' \| 'md' \| 'lg' \| 'xl'` | `'md'` | Margin around divider |
| `color` | `'default' \| 'primary' \| 'success' \| 'warning' \| 'error' \| 'info'` | `'default'` | Divider color |
| `className` | `string` | `''` | Additional CSS classes |
| `children` | `React.ReactNode` | - | Label text for labeled dividers |
