# Icon

A flexible icon component that supports custom SVG content and built-in icon names.

## Basic Usage

<CodeBlock code={`<Icon name="home" />
<Icon name="user" />
<Icon name="settings" />`}>
  <div className="flex items-center gap-4">
    <Icon name="home" />
    <Icon name="user" />
    <Icon name="settings" />
  </div>
</CodeBlock>
