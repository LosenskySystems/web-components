# Form Layout

A comprehensive form layout system with components for organizing and structuring forms with consistent spacing, labels, validation states, and error messages.

## Basic Usage

<CodeBlock code={`<Form onSubmit={handleSubmit}>
  <FormField label="Email" required>
    <Input type="email" placeholder="Enter your email" />
  </FormField>
  
  <FormField label="Password" required>
    <Input type="password" placeholder="Enter your password" />
  </FormField>
  
  <FormActions>
    <Button type="submit">Submit</Button>
  </FormActions>
</Form>`}>
  <div className="w-80">
    <Form onSubmit={(e) => e.preventDefault()}>
      <FormField label="Email" required>
        <Input type="email" placeholder="Enter your email" />
      </FormField>
      
      <FormField label="Password" required>
        <Input type="password" placeholder="Enter your password" />
      </FormField>
      
      <FormActions>
        <Button type="submit">Submit</Button>
      </FormActions>
    </Form>
  </div>
</CodeBlock>

## Complete Form Example

<CodeBlock code={`<Form onSubmit={handleSubmit}>
  <FormSection title="Account Setup" description="Create your account">
    <FormField label="Username" required errorText="Username is required" error>
      <Input placeholder="Choose a username" />
    </FormField>
    
    <FormField label="Email" required helperText="We'll use this for notifications">
      <Input type="email" placeholder="your@email.com" />
    </FormField>
    
    <FormField label="Password" required>
      <Input type="password" placeholder="Create a password" />
    </FormField>
  </FormSection>
  
  <FormSection title="Preferences">
    <FormField label="Newsletter">
      <Checkbox>Subscribe to newsletter</Checkbox>
    </FormField>
    
    <FormField label="Country">
      <Select>
        <Select.Item value="us">United States</Select.Item>
        <Select.Item value="ca">Canada</Select.Item>
        <Select.Item value="uk">United Kingdom</Select.Item>
      </Select>
    </FormField>
  </FormSection>
  
  <FormActions>
    <Button variant="ghost">Cancel</Button>
    <Button type="submit" variant="primary">Create Account</Button>
  </FormActions>
</Form>`}>
  <div className="w-96">
    <Form onSubmit={(e) => e.preventDefault()}>
      <FormSection title="Account Setup" description="Create your account">
        <FormField label="Username" required errorText="Username is required" error>
          <Input placeholder="Choose a username" />
        </FormField>
        
        <FormField label="Email" required helperText="We'll use this for notifications">
          <Input type="email" placeholder="your@email.com" />
        </FormField>
        
        <FormField label="Password" required>
          <Input type="password" placeholder="Create a password" />
        </FormField>
      </FormSection>
      
      <FormSection title="Preferences">
        <FormField label="Newsletter">
          <Checkbox>Subscribe to newsletter</Checkbox>
        </FormField>
        
        <FormField label="Country">
          <Select>
            <Select.Item value="us">United States</Select.Item>
            <Select.Item value="ca">Canada</Select.Item>
            <Select.Item value="uk">United Kingdom</Select.Item>
          </Select>
        </FormField>
      </FormSection>
      
      <FormActions>
        <Button variant="ghost">Cancel</Button>
        <Button type="submit" variant="primary">Create Account</Button>
      </FormActions>
    </Form>
  </div>
</CodeBlock>
