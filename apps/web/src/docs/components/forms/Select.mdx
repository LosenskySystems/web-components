# Select Component

The Select component provides a flexible, accessible dropdown selection interface with compound component architecture, various states, sizes, and styling options.

## Import

```tsx
import { Select, Label } from '@losensky-systems/web-components-core'
```

## Basic Usage

<div style={{ marginBottom: '2rem' }}>
  <Select placeholder="Choose an option...">
    <Select.Item value="option1">Option 1</Select.Item>
    <Select.Item value="option2">Option 2</Select.Item>
    <Select.Item value="option3">Option 3</Select.Item>
  </Select>
</div>

## Controlled Select

Use with state for controlled behavior:

<div style={{ marginBottom: '2rem' }}>
  <Select value="option2" placeholder="Choose an option...">
    <Select.Item value="option1">Option 1</Select.Item>
    <Select.Item value="option2">Option 2</Select.Item>
    <Select.Item value="option3">Option 3</Select.Item>
  </Select>
</div>

```tsx
const [value, setValue] = useState('option2')

<Select value={value} onChange={setValue} placeholder="Choose an option...">
  <Select.Item value="option1">Option 1</Select.Item>
  <Select.Item value="option2">Option 2</Select.Item>
  <Select.Item value="option3">Option 3</Select.Item>
</Select>
```

## Props

### Select Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Select size |
| `variant` | `'default' \| 'error' \| 'success' \| 'warning'` | `'default'` | Select variant |
| `error` | `boolean` | `false` | Error state |
| `errorText` | `string` | - | Error message |
| `successText` | `string` | - | Success message |
| `helperText` | `string` | - | Helper text |
| `loading` | `boolean` | `false` | Loading state |
| `disabled` | `boolean` | `false` | Disabled state |
| `required` | `boolean` | `false` | Required field |
| `autoFocus` | `boolean` | `false` | Auto-focus on mount |
| `placeholder` | `string` | `'Select an option...'` | Placeholder text |
| `value` | `string` | - | Selected value (controlled) |
| `onChange` | `(value: string) => void` | - | Change handler |
| `name` | `string` | - | Form field name |
| `id` | `string` | - | Element ID |
| `className` | `string` | `''` | Additional CSS classes |

### Select.Item Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `value` | `string` | - | Option value |
| `disabled` | `boolean` | `false` | Disabled state |
| `children` | `React.ReactNode` | - | Option content |
| `className` | `string` | `''` | Additional CSS classes |
