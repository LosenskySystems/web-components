# Input Component

The Input component is a flexible, accessible text input field with various states, sizes, and styling options.

## Import

```tsx
import { Input } from '@losensky-systems/web-components-core'
```

## Basic Usage

<CodeBlock code={`<Input placeholder="Enter your text..." />`}>
  <div className="mb-8">
    <Input placeholder="Enter your text..." />
  </div>
</CodeBlock>
