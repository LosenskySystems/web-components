# Input Component

The Input component is a flexible, accessible text input field with various states, sizes, and styling options.

## Import

```tsx
import { Input } from '@losensky-systems/web-components-core'
```

## Basic Usage

<div style={{ marginBottom: '2rem' }}>
  <Input placeholder="Enter your text..." />
</div>

```tsx
<Input placeholder="Enter your text..." />
```

    <Input

    />

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `type` | `'text' \| 'email' \| 'password' \| 'number' \| 'tel' \| 'url' \| 'search'` | `'text'` | Input type |
| `placeholder` | `string` | - | Placeholder text |
| `value` | `string` | - | Input value (controlled) |
| `defaultValue` | `string` | - | Default value (uncontrolled) |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Input size |
| `variant` | `'default' \| 'error' \| 'success' \| 'warning'` | `'default'` | Input variant |
| `error` | `boolean` | `false` | Error state |
| `errorText` | `string` | - | Error message |
| `successText` | `string` | - | Success message |
| `helperText` | `string` | - | Helper text |
| `icon` | `React.ReactNode` | - | Icon element |
| `iconPosition` | `'left' \| 'right'` | `'left'` | Icon position |
| `clearable` | `boolean` | `false` | Show clear button |
| `loading` | `boolean` | `false` | Loading state |
| `disabled` | `boolean` | `false` | Disabled state |
| `required` | `boolean` | `false` | Required field |
| `readOnly` | `boolean` | `false` | Read-only state |
| `autoComplete` | `string` | - | Autocomplete attribute |
| `autoFocus` | `boolean` | `false` | Auto-focus on mount |
| `maxLength` | `number` | - | Maximum length |
| `minLength` | `number` | - | Minimum length |
| `pattern` | `string` | - | Input pattern |
| `onChange` | `(event: React.ChangeEvent<HTMLInputElement>) => void` | - | Change handler |
| `onFocus` | `(event: React.FocusEvent<HTMLInputElement>) => void` | - | Focus handler |
| `onBlur` | `(event: React.FocusEvent<HTMLInputElement>) => void` | - | Blur handler |
| `className` | `string` | `''` | Additional CSS classes |
