# DateTimePicker

A comprehensive date and time picker component with calendar dropdown, time selection, and range support.

## Basic Usage

<CodeBlock code={`<DateTimePicker mode="date">Select a date</DateTimePicker>`}>
  <div className="w-64">
    <DateTimePicker mode="date">Select a date</DateTimePicker>
  </div>
</CodeBlock>

## Modes

### Date Only

<CodeBlock code={`<DateTimePicker mode="date" />`}>
  <div className="w-64">
    <DateTimePicker mode="date" />
  </div>
</CodeBlock>

### Time Only

<CodeBlock code={`<DateTimePicker mode="time" />`}>
  <div className="w-64">
    <DateTimePicker mode="time" />
  </div>
</CodeBlock>

### Date and Time

<CodeBlock code={`<DateTimePicker mode="datetime" />`}>
  <div className="w-80">
    <DateTimePicker mode="datetime" />
  </div>
</CodeBlock>

## Range Selection

<CodeBlock code={`<DateTimePicker rangeMode="range" />`}>
  <div className="w-80">
    <DateTimePicker rangeMode="range" />
  </div>
</CodeBlock>

## Controlled vs Uncontrolled

### Uncontrolled (default)

<CodeBlock code={`<DateTimePicker mode="date" defaultValue={new Date()} />`}>
  <div className="w-64">
    <DateTimePicker mode="date" defaultValue={new Date()} />
  </div>
</CodeBlock>

### Controlled

<CodeBlock code={`const [selectedDate, setSelectedDate] = useState(null);

<DateTimePicker 
  mode="date"
  value={selectedDate} 
  onChange={setSelectedDate}
/>`}>
  <div className="w-64">
    <DateTimePicker mode="date" />
  </div>
</CodeBlock>
