# Radio

A radio button component for single selection in forms.

## Basic Usage

<CodeBlock code={`<Radio name="option" value="option1">Option 1</Radio>`}>
  <Radio name="option" value="option1">Option 1</Radio>
</CodeBlock>

## Form Integration

<CodeBlock code={`// Basic Form Integration

  <Label>Payment Method</Label>

    <Radio name="payment-method" value="credit">Credit Card</Radio>
    <Radio name="payment-method" value="debit">Debit Card</Radio>
    <Radio name="payment-method" value="paypal">PayPal</Radio>
  </div>
</div>

// Required Selection
<div>
  <Label>Required Selection</Label>
  <div className="mt-2">
    <Radio name="required-field" value="agree" variant="error" onChange={(checked) => {
      if (!checked) {
        console.log('This field is required!');
      }
    }}>
      I agree to the terms and conditions
    </Radio>
  </div>
</div>`}>

        <Radio name="payment-method" value="paypal">PayPal</Radio>
      </div>
    </div>
