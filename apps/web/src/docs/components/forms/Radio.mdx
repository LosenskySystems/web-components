# Radio

A radio button component for single selection in forms.

## Basic Usage

<CodeBlock code={`<Radio name="option" value="option1">Option 1</Radio>`}>
  <Radio name="option" value="option1">Option 1</Radio>
</CodeBlock>

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | `false` | Whether the radio is checked |
| `disabled` | `boolean` | `false` | Whether the radio is disabled |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Radio size |
| `variant` | `'default' \| 'error' \| 'success'` | `'default'` | Radio variant |
| `children` | `React.ReactNode` | - | Radio label |
| `onChange` | `(checked: boolean) => void` | - | Callback when radio state changes |
| `className` | `string` | - | Additional CSS classes |
| `inputProps` | `React.InputHTMLAttributes<HTMLInputElement>` | - | Additional input props |
| `value` | `string` | - | Radio value for form handling |
| `name` | `string` | - | Radio name for grouping |
| `allowUncheck` | `boolean` | `false` | Allow unchecking the radio by clicking it again |
