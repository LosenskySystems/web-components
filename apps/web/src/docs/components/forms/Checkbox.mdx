# Checkbox

A checkbox component for form inputs and selections.

## Basic Usage

<CodeBlock code={`<Checkbox>Accept terms and conditions</Checkbox>`}>
  <Checkbox>Accept terms and conditions</Checkbox>
</CodeBlock>

## States

  <Checkbox checked disabled>Checked & Disabled</Checkbox>
</div>`}>

## Form Integration

<CodeBlock code={`// Basic Form Integration
<div>
  <Label>Preferences</Label>
  <div className="mt-2 space-y-2">
    <Checkbox>Email notifications</Checkbox>
    <Checkbox>SMS notifications</Checkbox>
    <Checkbox>Push notifications</Checkbox>
  </div>
</div>

// Required Field with Validation
<div>
  <Label>Required Field</Label>
  <div className="mt-2">
    <Checkbox variant="error" onChange={(checked) => {
      if (!checked) {
        console.log('This field is required!');
      }
    }}>
      I acknowledge the privacy policy
    </Checkbox>
  </div>
</div>`}>
  <div className="space-y-4">
    <div>
      <Label>Preferences</Label>
      <div className="mt-2 space-y-2">
        <Checkbox>Email notifications</Checkbox>
        <Checkbox>SMS notifications</Checkbox>
        <Checkbox>Push notifications</Checkbox>
      </div>
    </div>
    
    <div>
      <Label>Required Field</Label>
      <div className="mt-2">
        <Checkbox variant="error" onChange={(checked) => {
          if (!checked) {
            console.log('This field is required!');
          }
        }}>
          I acknowledge the privacy policy
        </Checkbox>
      </div>
    </div>
  </div>
</CodeBlock>
