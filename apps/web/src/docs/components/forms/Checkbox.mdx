## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `checked` | `boolean` | `false` | Whether the checkbox is checked |
| `indeterminate` | `boolean` | `false` | Whether the checkbox is in indeterminate state |
| `disabled` | `boolean` | `false` | Whether the checkbox is disabled |
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Checkbox size |
| `variant` | `'default' \| 'error' \| 'success'` | `'default'` | Checkbox variant |
| `children` | `React.ReactNode` | - | Checkbox label |
| `onChange` | `(checked: boolean) => void` | - | Callback when checkbox state changes |
| `className` | `string` | - | Additional CSS classes |
| `inputProps` | `React.InputHTMLAttributes<HTMLInputElement>` | - | Additional input props |
