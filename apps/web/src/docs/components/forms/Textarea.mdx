# Textarea Component

The Textarea component is a flexible, accessible multi-line text input with various states, sizes, styling options, and helper text support.

## Import

```tsx
import { Textarea, Label } from '@losensky-systems/web-components-core'
```

## Basic Usage

<CodeBlock code={`<Textarea placeholder="Enter your message..." />`}>
  <div className="mb-8">
    <Textarea placeholder="Enter your message..." />
  </div>
</CodeBlock>

## Form Integration

Textareas work well in forms with proper labels and validation:

<CodeBlock code={`<Label htmlFor="subject-input" required>Subject</Label>
<input id="subject-input" type="text" placeholder="Email subject" />

<Label htmlFor="message-textarea" required>Message</Label>
<Textarea 
  id="message-textarea" 
  rows={6} 
  placeholder="Write your message here..."
  resize="vertical"
  helperText="Minimum 10 characters required"
  minLength={10}
/>`}>
  <div className="mb-8">
    <div className="mb-4">
      <Label htmlFor="subject-input" required>Subject</Label>
      <input id="subject-input" type="text" placeholder="Email subject" />
    </div>
    <div>
      <Label htmlFor="message-textarea" required>Message</Label>
      <Textarea 
        id="message-textarea" 
        rows={6} 
        placeholder="Write your message here..."
        resize="vertical"
        helperText="Minimum 10 characters required"
        minLength={10}
      />
    </div>
  </div>
</CodeBlock>
