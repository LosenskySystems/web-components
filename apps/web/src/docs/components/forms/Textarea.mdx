# Textarea Component

The Textarea component is a flexible, accessible multi-line text input with various states, sizes, styling options, and helper text support.

## Import

```tsx
import { Textarea, Label } from '@losensky-systems/web-components-core'
```

## Basic Usage

<CodeBlock code={`<Textarea placeholder="Enter your message..." />`}>
  <div className="mb-8">
    <Textarea placeholder="Enter your message..." />
  </div>
</CodeBlock>

## Form Integration

Textareas work well in forms with proper labels and validation:

  id="message-textarea" 
  rows={6}

  minLength={10}

## Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `size` | `'sm' \| 'md' \| 'lg'` | `'md'` | Textarea size |
| `variant` | `'default' \| 'error' \| 'success' \| 'warning'` | `'default'` | Textarea variant |
| `error` | `boolean` | `false` | Error state |
| `errorText` | `string` | - | Error message |
| `successText` | `string` | - | Success message |
| `helperText` | `string` | - | Helper text |
| `resize` | `'none' \| 'vertical' \| 'horizontal' \| 'both'` | `'vertical'` | Resize behavior |
| `loading` | `boolean` | `false` | Loading state |
| `disabled` | `boolean` | `false` | Disabled state |
| `readOnly` | `boolean` | `false` | Read-only state |
| `required` | `boolean` | `false` | Required field |
| `autoFocus` | `boolean` | `false` | Auto-focus on mount |
| `maxLength` | `number` | - | Maximum length |
| `minLength` | `number` | - | Minimum length |
| `rows` | `number` | - | Number of visible text lines |
| `cols` | `number` | - | Number of visible character widths |
| `placeholder` | `string` | - | Placeholder text |
| `value` | `string` | - | Textarea value (controlled) |
| `defaultValue` | `string` | - | Default value (uncontrolled) |
| `onChange` | `(event: React.ChangeEvent<HTMLTextAreaElement>) => void` | - | Change handler |
| `onFocus` | `(event: React.FocusEvent<HTMLTextAreaElement>) => void` | - | Focus handler |
| `onBlur` | `(event: React.FocusEvent<HTMLTextAreaElement>) => void` | - | Blur handler |
| `className` | `string` | `''` | Additional CSS classes |
