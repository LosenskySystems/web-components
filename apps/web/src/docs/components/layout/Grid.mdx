# Grid

A flexible CSS Grid-based layout system with responsive column support, gap spacing, and alignment options for building complex layouts.

## Basic Usage

<CodeBlock code={`<Grid cols={3} gap="md">
  <Grid.Item>Item 1</Grid.Item>
  <Grid.Item>Item 2</Grid.Item>
  <Grid.Item>Item 3</Grid.Item>
  <Grid.Item>Item 4</Grid.Item>
  <Grid.Item>Item 5</Grid.Item>
  <Grid.Item>Item 6</Grid.Item>
</Grid>`}>
  <Grid cols={3} gap="md">
    <Grid.Item>
      <div className="bg-blue-100 p-4 rounded text-center">Item 1</div>
    </Grid.Item>
    <Grid.Item>
      <div className="bg-blue-100 p-4 rounded text-center">Item 2</div>

## Polymorphic Component

<CodeBlock code={`<Grid as="section" cols={2} gap="md">
  <Grid.Item as="article">
    <div className="bg-gray-100 p-4 rounded">
      <h3 className="font-semibold mb-2">Article 1</h3>
      <p>This Grid renders as a section element.</p>
    </div>
  </Grid.Item>
  <Grid.Item as="article">
    <div className="bg-gray-100 p-4 rounded">
      <h3 className="font-semibold mb-2">Article 2</h3>
      <p>Grid items render as article elements.</p>
    </div>
  </Grid.Item>
</Grid>`}>
  <Grid as="section" cols={2} gap="md">
    <Grid.Item as="article">
      <div className="bg-gray-100 p-4 rounded">
        <h3 className="font-semibold mb-2 text-gray-900">Article 1</h3>
        <p className="text-gray-600 text-sm">This Grid renders as a section element.</p>
      </div>
    </Grid.Item>
    <Grid.Item as="article">
      <div className="bg-gray-100 p-4 rounded">
        <h3 className="font-semibold mb-2 text-gray-900">Article 2</h3>
        <p className="text-gray-600 text-sm">Grid items render as article elements.</p>
      </div>
    </Grid.Item>
  </Grid>
</CodeBlock>

### GridProps

  { name: 'cols', type: "number | { sm?: number; md?: number; lg?: number; xl?: number; '2xl'?: number }", default: '1', description: 'Number of columns or responsive column configuration.' },
  { name: 'rows', type: "number | 'auto'", description: 'Number of rows (optional).' },
  { name: 'gap', type: "'none' | 'xs' | 'sm' | 'md' | 'lg' | 'xl'", description: 'Gap spacing for both directions.' },
  { name: 'gapX', type: "'none' | 'xs' | 'sm' | 'md' | 'lg' | 'xl'", description: 'Horizontal gap spacing only.' },
  { name: 'gapY', type: "'none' | 'xs' | 'sm' | 'md' | 'lg' | 'xl'", description: 'Vertical gap spacing only.' },
  { name: 'align', type: "'start' | 'center' | 'end' | 'stretch'", description: 'Align items on the cross axis.' },
  { name: 'justify', type: "'start' | 'center' | 'end' | 'stretch' | 'between' | 'around'", description: 'Justify items on the main axis.' },
  { name: 'as', type: 'React.ElementType', default: "'div'", description: 'HTML element to render as.' },
  { name: 'className', type: 'string', description: 'Additional CSS classes.' },

]} />
