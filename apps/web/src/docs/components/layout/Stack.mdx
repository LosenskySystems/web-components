# Stack

A flexible layout component for arranging children vertically or horizontally with consistent spacing and alignment options.

## Basic Usage

<CodeBlock code={`<Stack spacing="md">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</Stack>`}>
  <Stack spacing="md">
    <div className="bg-blue-100 p-4 rounded">Item 1</div>
    <div className="bg-blue-100 p-4 rounded">Item 2</div>
    <div className="bg-blue-100 p-4 rounded">Item 3</div>
  </Stack>
</CodeBlock>

## Polymorphic Component

  </article>
  <article className="bg-gray-100 p-4 rounded">

    <article className="bg-gray-100 p-4 rounded">
      <h3 className="font-semibold mb-2 text-gray-900">Article 1</h3>
      <p className="text-gray-600 text-sm">This Stack renders as a section element.</p>
    </article>
    <article className="bg-gray-100 p-4 rounded">
